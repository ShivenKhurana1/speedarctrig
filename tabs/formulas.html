Trigonometry Reference

Basic Trig Functions

sin(θ) = opposite/hypotenuse

Sine function

cos(θ) = adjacent/hypotenuse

Cosine function

tan(θ) = opposite/adjacent

Tangent function

Reciprocal Trig Functions

csc(θ) = 1/sin(θ)

Cosecant

sec(θ) = 1/cos(θ)

Secant

cot(θ) = 1/tan(θ)

Cotangent

Inverse Trig Functions

arcsin(x) = θ where sin(θ) = x

Arcsine (range: -π/2 to π/2)

arccos(x) = θ where cos(θ) = x

Arccosine (range: 0 to π)

arctan(x) = θ where tan(θ) = x

Arctangent (range: -π/2 to π/2)

Pythagorean Identities

sin²(θ) + cos²(θ) = 1

1 + tan²(θ) = sec²(θ)

1 + cot²(θ) = csc²(θ)
